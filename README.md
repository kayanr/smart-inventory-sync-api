
## Smart Inventory Sync API

A backend service built with FastAPI that manages inventory data and synchronizes supplier data through a custom integration pipeline.

### Tech Stack

* Python 3.12
* FastAPI
* SQLAlchemy
* SQLite (initial)
* Docker
* Pytest
* GitHub Actions (CI)

### Features (Planned)

* Inventory CRUD API
* Supplier data sync and transformation
* Data pipeline processing
* Containerized deployment
* Automated testing and CI pipeline
